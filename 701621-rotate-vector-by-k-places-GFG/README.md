# [Rotate vector by K places](https://www.geeksforgeeks.org/problems/rotate-vector-by-k-places/1)
## Easy
Given a vector arr[]&nbsp;and an integer k. You need to left rotate the vector by k places.
Examples:
Input: arr[] = [1, 2, 3, 4], k = 3Output: [4, 1, 2, 3]Explanation: arr[] is rotated by 3 places.
Input: arr[] = [4, 3, 2, 1, 0], k = 2Output: [2, 1, 0, 4, 3]Explanation: arr[] is rotated by 2 places.
Constraints:1 &lt;= arr.size()&nbsp;&lt;= 10001 &lt;= arr[i]&nbsp;&lt;= 10001 &lt;= k&nbsp;&lt;= 1000
